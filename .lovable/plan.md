# הפיכת המערכת לפשוטה ונעימה — עד 2 קליקים לכל פעולה

## העקרונות המנחים
1. **כל פעולה = עד 2 קליקים** מהמסך הראשי.
2. **אין "צפייה" נפרדת מ"עריכה"** — הטקסט תמיד עריך במקום, שמירה אוטומטית.
3. **כפתור ראשי אחד גדול** בכל מסך (Primary Action) — היתר הולך לתפריט "עוד".
4. **פידבק מיידי** — toast עדין, אנימציות קלות, מצבי טעינה ברורים.
5. **שפה אחידה ונעימה** — אייקונים עקביים, צבעים רכים, מרווחים נדיבים.

## 1. עריכת תמלול בקליק אחד (TranscriptViewerDialog)

**עכשיו:** קליק על "צפייה" → טאב "צפייה" → צריך לעבור לטאב "עריכה" → לערוך → לשמור.

**אחרי:**
- קליק אחד על שם ההקלטה / על כרטיס ההקלטה → נפתח **עורך אחד מאוחד** (בלי טאבים).
- הטקסט מוצג בתוך `Textarea` גדול ונעים — **עריך מיד**, בלי מצב "תצוגה".
- **שמירה אוטומטית** (debounce 1.5 שניות) + אינדיקטור עדין "נשמר ✓".
- סרגל צד מתקפל אחד עם **"החלפות"** (דוברים + מילות מפתח יחד) — כפתור "החלף הכל" שמיר.
- היסטוריית גרסאות + תמלול-מחדש = תפריט "⋯ עוד" בפינה.
- כפתורי ייצוא (PDF/TXT/העתק) = אייקונים קטנים בכותרת, לא כפתורים גדולים.

```text
┌─────────────────────────────────────────────┐
│  שם_הקובץ.mp3      📋 ⬇️ 📄  ⋯  ✕  │
│  נשמר ✓ · תמלול-על                          │
├──────────────────┬──────────────────────────┤
│                  │  החלפות (3)        ▾    │
│   טקסט עריך      │  דובר 1 → דנה            │
│   ישירות כאן     │  דובר 2 → יוסי           │
│   ...            │  [+ הוסף החלפה]          │
│                  │  [החלף הכל]              │
└──────────────────┴──────────────────────────┘
```

## 2. כרטיס הקלטה פשוט (Recordings / MeetingRecordings)

**עכשיו:** הרבה כפתורים בשורה — תמלל / צפייה / שייך / dropdown.

**אחרי — כל כרטיס:**
- **קליק על הכרטיס עצמו** = פעולה ראשית חכמה:
  - אם יש תמלול → פותח עורך.
  - אם אין תמלול → פותח דיאלוג בחירה (מהיר / על).
- **אייקון ⋯ אחד בפינה** עם: שייך, פתח ב-Drive, הורד, מחק.
- סטטוס = badge צבעוני קטן (ממתין=אפור, מתמלל=כחול עם spinner, הושלם=ירוק, שגיאה=אדום).
- תאריך + משך = שורה אחת קטנה מתחת לשם.

## 3. ניווט עד 2 קליקים

מהמסך הראשי (RoleHome) — כל יעד נגיש בקליק אחד מהסיידבר. הפעולה עצמה = קליק שני.

| מטרה | קליק 1 | קליק 2 |
|---|---|---|
| לערוך תמלול | "הקלטות" בסיידבר | קליק על הכרטיס |
| לתמלל קובץ חדש | "הקלטות" | "תמלל" על הכרטיס |
| להקליט שיחה | כפתור "🎙️ הקלט" צף | "התחל" |
| לסכם פגישה | "פגישות" | "סכם" על הכרטיס |
| לסנכרן Drive | באנר עליון | "סנכרן" |

## 4. שיפורי UX/עיצוב

- **כפתור פעולה צף (FAB)** בפינה ימין-תחתון בכל מסך רשימה — הפעולה הראשית של המסך (הקלט / העלה / צור פגישה).
- **Empty States חמים** עם איור + משפט עידוד + כפתור יחיד ברור.
- **Toast עדין בעברית** במקום הודעות טכניות — "התמלול הסתיים בהצלחה ✓".
- **Skeleton loaders** במקום ספינר ריק.
- **גופן + מרווחים**: הגדלת padding בכרטיסים, line-height נדיב יותר, פחות גבולות חדים — יותר shadow רך.
- **צבעים רכים יותר**: מעבר לפלטה פסטלית עדינה לתגיות סטטוס (כיום הצבעים חזקים מדי).
- **Tooltip עברי** על כל אייקון שאינו טקסט.

## 5. פעולות מאוחדות (פחות דיאלוגים)

- **TranscribeDialog**: במקום שני כפתורים גדולים "מהיר/על" — Toggle אחד עם הסבר קצר מתחת, וכפתור "התחל תמלול" יחיד.
- **AssignRecordingDialog**: חיפוש מיידי + רשימה — קליק על תיק = שיוך מיידי וסגירה (בלי "אישור").
- **MergeTranscriptsDialog**: להעלים — המיזוג קורה אוטומטית בסיום "תמלול-על".
